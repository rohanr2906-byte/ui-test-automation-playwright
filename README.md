# UI Test Automation Demo using Playwright

This is a self-developed UI test automation demo project using Python, Playwright, pytest, and pytest-html.

## Project Objective

The objective of this project is to demonstrate basic UI test automation skills by automating a login scenario on a demo web application.

## Tools Used

- Python
- Playwright
- pytest
- pytest-html
- Git
- GitHub

## Test Scenario

### Login Test

- Opens the SauceDemo website
- Enters username and password
- Clicks the login button
- Verifies that the Products page is displayed after successful login
